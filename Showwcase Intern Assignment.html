User Engagement Analysis - Data Analyst @ Showwcase¶

This sheet will help to understand the "user engagement" on the platform for the month of October, 2019. Based on the given dataset, the a set of analytics based on different parameters have been generated.

To better understand the project and the process, it has been divided into three sections: 1) Data Loading 2) Data Cleaning and Transformation 3) Data Analysis
